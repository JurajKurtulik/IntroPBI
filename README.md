# Introduction to Power BI

A dark, interactive, scroll-driven website presentation built from `Introduction-to-Power-BI.pptx`.

## Run locally

```bash
python3 -m http.server 8888
```

Then open:

```text
http://127.0.0.1:8888/
```

## GitHub Pages

This is a static site with no build step. Publish it from the repository root on the `main` branch.

Expected Pages URL:

```text
https://jurajkurtulik.github.io/IntroPBI/
```

All UI, animations, and interactions are contained in `index.html`, `styles.css`, and `app.js`.
